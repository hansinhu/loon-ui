# loon-ui-cli

> loonxi vue ui

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
# Development
npm run dev

# build Components for lib
npm run build:lib

# build theme for lib
npm run build:theme

# build doc for lib
npm run build:doc

```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
